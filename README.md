# TrainScheduleManagementSystem

This is a repo for Train Schedule Management System developed using Python.



It gives you an option to choose from all below functionalities. You can -

1\. Add a new train schedule and save it into trains.txt.

2\. Display all train schedules stored in the file.

3\. Search for a train using its train number.

4\. Search trains based on source and destination stations.

5\. Update the departure and arrival time of a train.

6\. Delete a train schedule using the train number.



This system also prevents duplicate train numbers from being added.
